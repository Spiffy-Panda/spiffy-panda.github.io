# 🛠 Building with Agents

Short pitches on projects I build *with* AI agents — each framed by **how** Claude was used (coder, orchestrator, data-cruncher, in-the-loop operator), not just what got built. Pick whichever sparks your interest.

- [[builder-ideas/ai-verbal-coaching-builder|ai-verbal-coaching]]: (In chat client workflow) Claude as the live runtime client of an MCP server it also built; coaches a spoken pitch, ums and all.
- [[builder-ideas/DemonBluffAi-Redux-builder|DemonBluffAi-Redux]]: (Build a tool and a lesson of how it works) a computer-vision course Claude wrote end to end (incl. a GPU fine-tune) — a teaching artifact about disciplined agent collaboration.
- [[builder-ideas/ProductivityAndReadingHelper-builder|ProductivityAndReadingHelper]]: (Natural Language Transformations) Claude as overnight orchestrator and *gatekeeper* of a local paper-to-podcast study pipeline.
- [[builder-ideas/AI-BT-Gym-builder|AI-BT-Gym]]: (Visualizers, Data Creation and Analysis) Claude as the genetic operator of an evolutionary behavior-tree gym, mutating game AI from match telemetry.
- [[builder-ideas/OstranautDataExplorer-builder|OstranautDataExplorer]]: (Data Store Explorer, Support Desk Workflow) Claude builds a "Find Usages" graph explorer for an opaque game-data tree; reusable agent-workflow patterns fall out.
- [[builder-ideas/SoundingChamber-builder|SoundingChamber]]: (Data Web Collection and HITL TTS Tuning) Claude + parallel subagents stand up a linguistics puzzle game fast: 181 sound-change rules and a custom TTS package.
